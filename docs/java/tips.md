- Check if the number (a) is even or odd

    ``` boolean = (a & 1) == 0 ? "Even" : "ODD" ```
- Fast multiplication number (n) by factor 2
    
    ``` n = n << 1 ```
- Fast division number (n) by factor 2
    
    ``` n = n >> 1 ```
